# Proteome English Words Analysis

## Overview
A Python application that analyzes the presence of English words within a human proteome (FASTA file). It features a GUI built with Tkinter and provides statistical insights into word frequency and distribution across protein sequences.

## Features
- Reads and filters word lists using biological constraints (AA_LETTERS)
- Processes FASTA files efficiently
- Performs dual-frequency analysis (by protein count and total occurrences)
- Identifies most frequent motifs
- User-friendly graphical interface (Tkinter)
